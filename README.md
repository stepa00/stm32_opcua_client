# STM32 OPC UA project

The most basic project of opc ua client using open62541 library on Necleo-F767zi board.

OPC UA library set on FREERTOS + LWIP packages. 

Project contains a complete [tutorial](tutorial/tutorial_v2.md) with sources to recreate and improve the project.

To setup project on your machine:

1. Create a new workspace in STM32 CubeIDE.
2. Clone main repository into workspace.
3. Open project **File > Open Projects from File System...**.
